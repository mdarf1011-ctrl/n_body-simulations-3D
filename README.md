# n_body-simulations-3D
## 🛰️ 3D Two-Body Orbital Mechanics Simulation
<img width="1536" height="756" alt="n_body_sim" src="https://github.com/user-attachments/assets/572f4f28-c7ca-4fc8-a2c8-349394ef0749" />
A high-fidelity numerical physics engine built in Python utilizing **Symplectic Euler Integration** to simulate a satellite orbiting a rotating Earth.

### 🛠️ Features & Telemetry
* **Dynamic Local Reference Frames:** Computes real-time **RTN (Radial, Tangential, Normal)** coordinate basis tripods on the satellite using continuous vector cross products ($\vec{R} \times \vec{V}$).
* **Synchronized Planetary Rotation:** Features a rotating Earth wireframe mesh mapped to the planet's true angular velocity ($\omega_{earth} = 7.2921 \times 10^{-5} \text{ rad/s}$).
